# Repository for UWB-based sensors

This repository contains the code for the UWB-based sensors developed by the team at the Innopolis University.

## Contents

The repository contains the following branches:

| Branch | Description | Link |
| ------ | ----------- | ---- |
| main | Contains the code for the sensors hardware. | [branch](https://github.com/Innopolis-UAV-Team/uwb_beacons/tree/main)|
| ros1 | Contains the code for ROS1 node | [branch](https://github.com/Innopolis-UAV-Team/uwb_beacons/tree/ROS1)|
| ros2 | Contains the code for ROS2 node | [branch](https://github.com/Innopolis-UAV-Team/uwb_beacons/tree/ros2)|
